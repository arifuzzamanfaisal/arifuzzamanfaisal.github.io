A prime number is a positive integer larger than 1 that has no additional factors other than 1 and the number itself. The numbers 2, 3, 5, 7, 11, etc. are prime numbers since there are no other factors. To discover a prime number in Python, the value must be iterated from beginning to finish with a for loop and for each count, verify whether it divides n and change the flag value from 1 to 0. If predefined flag is 1 then the number is Prime. else the number is not Prime. 

num = int(input("Input a number : "))
flag = 1

if num>1:
    for n in range(2,num):
        if num%n==0:
            flag = 0
            break
    
if flag == 1 or num == 2:
    print("The number is prime")
else:
    print("The number is not prime")
